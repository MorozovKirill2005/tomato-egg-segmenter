# Изучение методов классического CV на примере задачи сегментации и классификации помидоров и яиц

В данной работе проводится изучение способов обработки изображений на основе точечных и пространственных преобразований. Применение различных методов обработки из классического CV рассматривается на задаче сегментации объектов (помидоров Черри 2-х видов и перепелиных яиц) на изображении. Задание заключается в разработке алгоритма обработки входного изображения, содержащего объекты перечисленных выше классов, который осуществляет сегментацию каждого объекта, классификацию и подсчёт представителей каждого класса на входной картинке. Для удобной работы с алгоритмом реализуется приложение с возможностью загрузки произвольного изображения, визуализацией результатов работы алгоритма и возможностью сохранения полученного изображения.

Подробнее о задании и способе его решения написано в файле <<Отчёт.pdf>>.

## Быстрый старт

Для того, чтобы протестировать реализованный алгоритм, выполните следующие действия:

```bash
# Клонирование
git clone https://github.com/MorozovKirill2005/tomato-egg-segmenter.git
cd tomato-egg-segmenter

# Установка зависимостей
pip install -r requirements.txt

# Запуск программы
python app.py
```
